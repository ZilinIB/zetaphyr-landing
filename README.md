# ZetaAI Landing Page

Coming soon page for ZetaAI - an agentic video editing tool.

## Development

```bash
cd zeta-ai
npm install
npm run dev
```

## Build

```bash
cd zeta-ai
npm run build
```

## Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm install -g vercel
cd zeta-ai
vercel
```

### Option 2: Vercel Dashboard
1. Push code to GitHub
2. Import project in Vercel dashboard
3. Set root directory to `zeta-ai`
4. Deploy

The project is pre-configured with `vercel.json` for seamless deployment.
